Dutch Dictionary
Woordenboek lets you find the definition of Dutch words. Type a word and all the dictionary information will be displayed.
Search results are stored and available offline.
The application is in English buy only the definition of Dutch words can be found.

An internet connection is required.

New in 1.1:
History function to see last searches.
